Normal Distribution

иҰҶзӣ– QR/QT йқўиҜ•дёӯжӯЈжҖҒеҲҶеёғзӣёе…ізҡ„ж ёеҝғз»“и®әгҖҒжҺЁеҜјж–№жі•дёҺз»Ҹе…ёйўҳеһӢпјҢйҮҚзӮ№еҢ…еҗ«и§’еәҰе…¬ејҸгҖҒжқЎд»¶жңҹжңӣгҖҒз”ҹжҲҗж–№жі•дёҺиҒ”еҗҲжӯЈжҖҒгҖӮ

01

еҹәжң¬жҖ§иҙЁдёҺе…ій”®е…¬ејҸ

ж ҮеҮҶжӯЈжҖҒ \(Z \sim \mathcal{N}(0,1)\)
$$\phi(z) = \frac{1}{\sqrt{2\pi}}e^{-z^2/2}$$ $$\Phi(x) = \int_{-\infty}^{x}\phi(t)\,dt$$

еҜ№з§°жҖ§пјҡ\(\phi(-z)=\phi(z)\)пјҢ\(\Phi(-x)=1-\Phi(x)\)

зәҝжҖ§еҸҳжҚў
$$X \sim \mathcal{N}(\mu,\sigma^2)$$ $$aX + b \sim \mathcal{N}(a\mu+b,\; a^2\sigma^2)$$

зӢ¬з«ӢжӯЈжҖҒд№Ӣе’ҢпјҡиӢҘ \(X_i \sim \mathcal{N}(\mu_i, \sigma_i^2)\) зӢ¬з«ӢпјҢ

$$\sum X_i \sim \mathcal{N}\!\left(\sum\mu_i,\sum\sigma_i^2\right)$$
йҮҚиҰҒзҹ©дёҺзӯүејҸ
еҘҮж•°зҹ©
$$\mathbb{E}[Z^{2k+1}] = 0$$
еҒ¶ж•°зҹ©
$$\mathbb{E}[Z^{2k}] = (2k-1)!!$$
е…ій”®зӯүејҸ
$$\mathbb{E}[Z^2] = 1,\;\mathbb{E}[Z^4] = 3$$
ж ҮеҮҶжӯЈжҖҒеҪ’дёҖеҢ–иҜҒжҳҺпјҲжһҒеқҗж Үпјү
$$I^2 = \int_{-\infty}^{\infty}e^{-x^2/2}dx\cdot\int_{-\infty}^{\infty}e^{-y^2/2}dy = \int_0^{2\pi}\int_0^{\infty}e^{-r^2/2}r\,dr\,d\theta = 2\pi$$
жҲӘж–ӯжӯЈжҖҒ \(\mathbb{E}[Z \mid Z > 0]\)
$$\mathbb{E}[Z \mid Z > 0] = \sqrt{\frac{2}{\pi}} \approx 0.7979$$
з”ұеҜ№з§°жҖ§ \(P(Z>0)=\tfrac{1}{2}\)
\(\mathbb{E}[Z \cdot \mathbf{1}_{Z>0}] = \int_0^\infty z\frac{1}{\sqrt{2\pi}}e^{-z^2/2}dz = \frac{1}{\sqrt{2\pi}}\)
\(\mathbb{E}[Z \mid Z>0] = \dfrac{\mathbb{E}[Z\cdot\mathbf{1}_{Z>0}]}{P(Z>0)} = \dfrac{1/\sqrt{2\pi}}{1/2} = \sqrt{\dfrac{2}{\pi}}\)
рҹ’Ў
дёҖиҲ¬жғ…еҪўпјҡиӢҘ \(X \sim \mathcal{N}(\mu,\sigma^2)\)пјҢеҲҷ \(\mathbb{E}[X\mid X>0] = \mu + \sigma\dfrac{\phi(-\mu/\sigma)}{\Phi(\mu/\sigma)}\)пјҲMills ratioпјү
02

иҒ”еҗҲжӯЈжҖҒеҲҶеёғ

е®ҡд№үпјҲзӯүд»·еҲ»з”»пјү

\((X,Y)\) жҳҜиҒ”еҗҲжӯЈжҖҒеҪ“дё”д»…еҪ“еҜ№д»»ж„Ҹеёёж•° \(a,b\)пјҢ\(aX+bY\) жңҚд»ҺпјҲдёҖе…ғпјүжӯЈжҖҒеҲҶеёғгҖӮ

вҡ пёҸ
жіЁж„Ҹпјҡиҫ№зјҳжӯЈжҖҒ вү  иҒ”еҗҲжӯЈжҖҒгҖӮеӯҳеңЁ \(X,Y\) еҗ„иҮӘжңҚд»Һ \(\mathcal{N}(0,1)\) дҪҶиҒ”еҗҲеҲҶеёғдёҚжҳҜжӯЈжҖҒзҡ„дҫӢеӯҗпјҲеҰӮ \(Y=X\cdot\text{sgn}(W)\)пјҢ\(W\perp X\)пјүгҖӮ
дәҢз»ҙжӯЈжҖҒзҡ„ PDFпјҲ\(\mathbf{n=2}пјү
$$f(x,y) = \frac{1}{2\pi\sigma_X\sigma_Y\sqrt{1-\rho^2}} \exp\!\left(-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_X)^2}{\sigma_X^2} - \frac{2\rho(x-\mu_X)(y-\mu_Y)}{\sigma_X\sigma_Y} + \frac{(y-\mu_Y)^2}{\sigma_Y^2}\right]\right)$$
д»ҝе°„еҸҳжҚў

иӢҘ \(\mathbf{X} \sim \mathcal{N}(\boldsymbol{\mu}, \Sigma)\)пјҢ\(\mathbf{Y} = c + B\mathbf{X}\)пјҢеҲҷ

$$\mathbf{Y} \sim \mathcal{N}(c + B\boldsymbol{\mu},\; B\Sigma B^T)$$

зәҝжҖ§еҸҳжҚўе°ҶиҒ”еҗҲжӯЈжҖҒжҳ е°„еҲ°иҒ”еҗҲжӯЈжҖҒпјҢеҚҸж–№е·®зҹ©йҳөжҢү \(B\Sigma B^T\) еҸҳжҚўгҖӮ

иҒ”еҗҲжӯЈжҖҒзҡ„зӢ¬з«ӢжҖ§еҲӨе®ҡ
$$\text{иҒ”еҗҲжӯЈжҖҒдёӯпјҡ}\quad X \perp Y \iff \text{Cov}(X,Y) = 0$$
рҹ’Ў
иҝҷжҳҜиҒ”еҗҲжӯЈжҖҒзҡ„зү№ж®ҠжҖ§иҙЁвҖ”вҖ”дёҖиҲ¬жғ…еҶөдёӢдёҚзӣёе…ідёҚзӯүдәҺзӢ¬з«ӢпјҢдҪҶиҒ”еҗҲжӯЈжҖҒдёӯдәҢиҖ…зӯүд»·гҖӮйқўиҜ•й«ҳйў‘иҖғзӮ№гҖӮ
03

жқЎд»¶еҲҶеёғ

дәҢз»ҙиҒ”еҗҲжӯЈжҖҒзҡ„жқЎд»¶еҲҶеёғ

и®ҫ \((X,Y)\sim\mathcal{N}\bigl((\mu_X,\mu_Y), \begin{pmatrix}\sigma_X^2 & \rho\sigma_X\sigma_Y \\ \rho\sigma_X\sigma_Y & \sigma_Y^2\end{pmatrix}\bigr)\)пјҢз»ҷе®ҡ \(X=x\)пјҡ

$$Y \mid X=x \;\sim\; \mathcal{N}\!\left(\mu_Y + \rho\frac{\sigma_Y}{\sigma_X}(x-\mu_X),\;\; \sigma_Y^2(1-\rho^2)\right)$$

е…ій”®и®°еҝҶпјҡжқЎд»¶еқҮеҖјжҳҜ \(x\) зҡ„зәҝжҖ§еҮҪж•°пјӣжқЎд»¶ж–№е·® \(\sigma_Y^2(1-\rho^2)\) дёҺ \(x\) ж— е…ігҖӮ

зӯүд»·жһ„йҖ пјҲCholesky и§’еәҰпјү

д»Ө \(Z_1,Z_2 \overset{iid}{\sim} \mathcal{N}(0,1)\)пјҢжһ„йҖ пјҡ

$$X = \mu_X + \sigma_X Z_1, \qquad Y = \mu_Y + \rho\sigma_Y Z_1 + \sqrt{1-\rho^2}\,\sigma_Y Z_2$$

еҲҷ \((X,Y)\) жҳҜе…·жңүзӣёе…ізі»ж•° \(\rho\) зҡ„иҒ”еҗҲжӯЈжҖҒгҖӮз»ҷе®ҡ \(X=x\) еҚіеӣәе®ҡ \(Z_1 = \frac{x-\mu_X}{\sigma_X}\)пјҢ\(Y\mid X=x\) дёӯзҡ„йҡҸжңәжҖ§жқҘиҮӘ \(\sqrt{1-\rho^2}\,\sigma_Y Z_2\)гҖӮ

рҹ’Ў
иҝҷдёӘжһ„йҖ и®©жқЎд»¶еҲҶеёғзҡ„з»“жһңеҸҳеҫ—зӣҙи§Ӯпјҡ
\(\mathbb{E}[Y\mid X=x] = \mu_Y + \rho\sigma_Y \cdot \frac{x-\mu_X}{\sigma_X}\)пјҢ
\(\mathrm{Var}(Y\mid X=x) = (1-\rho^2)\sigma_Y^2\)гҖӮ
еӨҡз»ҙжғ…еҪўпјҲй«ҳз»ҙиҒ”еҗҲжӯЈжҖҒпјү

и®ҫ \(\mathbf{X} = \begin{pmatrix}\mathbf{X}_1\\\mathbf{X}_2\end{pmatrix} \sim \mathcal{N}\!\left(\begin{pmatrix}\boldsymbol{\mu}_1\\\boldsymbol{\mu}_2\end{pmatrix}, \begin{pmatrix}\Sigma_{11} & \Sigma_{12}\\\Sigma_{21} & \Sigma_{22}\end{pmatrix}\right)\)пјҢеҲҷ

$$\mathbf{X}_1 \mid \mathbf{X}_2 = \mathbf{x}_2 \;\sim\; \mathcal{N}\!\left(\boldsymbol{\mu}_1 + \Sigma_{12}\Sigma_{22}^{-1}(\mathbf{x}_2 - \boldsymbol{\mu}_2),\;\; \Sigma_{11} - \Sigma_{12}\Sigma_{22}^{-1}\Sigma_{21}\right)$$

еҗҺдёҖйЎ№ \(\Sigma_{11} - \Sigma_{12}\Sigma_{22}^{-1}\Sigma_{21}\) жҳҜ Schur иЎҘпјҢеҚіжқЎд»¶еҚҸж–№е·®зҹ©йҳөгҖӮ

04

и§’еәҰе…¬ејҸдёҺеҚҠе№ійқўжҰӮзҺҮ

ж ёеҝғе®ҡзҗҶ

и®ҫ \((U,V)\) жҳҜеқҮеҖјдёәйӣ¶зҡ„иҒ”еҗҲжӯЈжҖҒпјҢзӣёе…ізі»ж•°дёә \(\rho_{UV}\)пјҢеҲҷ

$$\boxed{P(U>0,\, V>0) = \frac{1}{4} + \frac{1}{2\pi}\arcsin(\rho_{UV})}$$
жҺЁеҜјпјҡд»Һи§’еәҰзҗҶи§Ј
еҜ№йӣ¶еқҮеҖјиҒ”еҗҲжӯЈжҖҒпјҢдҪңж ҮеҮҶеҢ–еҸҳжҚўпјҡд»Ө \(\tilde{U} = U/\sigma_U,\; \tilde{V} = V/\sigma_V\)пјҢд»ҚжҳҜзӣёе…ізі»ж•° \(\rho\) зҡ„иҒ”еҗҲжӯЈжҖҒпјҲдё”ж–№е·®еқҮдёә1пјүгҖӮ
йҖҡиҝҮ Cholesky еҲҶи§ЈпјҢд»Ө \(\tilde{U} = Z_1\)пјҢ\(\tilde{V} = \rho Z_1 + \sqrt{1-\rho^2}Z_2\)пјҢе…¶дёӯ \(Z_1,Z_2 \overset{iid}{\sim}\mathcal{N}(0,1)\)гҖӮ
е°Ҷ \((Z_1,Z_2)\) и§ҶдҪңе№ійқўдёҠзҡ„еқҮеҢҖйҡҸжңәж–№еҗ‘пјҢиҪ¬еҢ–дёәжһҒеқҗж Ү \((r,\theta)\)пјҢ\(\theta\sim\text{Uniform}[0,2\pi)\)гҖӮ
жқЎд»¶ \(\{\tilde{U}>0,\tilde{V}>0\}\) зӯүд»·дәҺ \(Z_1>0\) дё” \(\rho Z_1+\sqrt{1-\rho^2}Z_2>0\)пјҢеҮ дҪ•дёҠеҜ№еә”ж–№еҗ‘и§’иҗҪеңЁжҹҗдёҖжүҮеҪўеҶ…гҖӮ
жүҮеҪўи§’еәҰ = \(\dfrac{\pi/2 + \arcsin\rho}{2\pi}\cdot 2\pi = \dfrac{\pi}{2}+\arcsin\rho\)пјҢ
жҰӮзҺҮ = \(\dfrac{\pi/2+\arcsin\rho}{2\pi} = \dfrac{1}{4} + \dfrac{\arcsin\rho}{2\pi}\)гҖӮ
жҺЁе№ҝе…¬ејҸ
дёӨдёӘеҚҠз©әй—ҙ
$$P(U>0, V>0) = \frac{1}{4} + \frac{\arcsin\rho}{2\pi}$$
дёҖжӯЈдёҖиҙҹ
$$P(U>0, V<0) = \frac{1}{4} - \frac{\arcsin\rho}{2\pi}$$
\(\rho=0\)пјҲзӢ¬з«Ӣпјү
$$P = \frac{1}{4}$$
\(\rho=1\)пјҲе®Ңе…ЁжӯЈзӣёе…іпјү
$$P = \frac{1}{4}+\frac{1}{4}=\frac{1}{2}$$
рҹ“җ
еҮ дҪ•зӣҙи§үпјҡ\(\rho = \cos\theta\) е…¶дёӯ \(\theta\) жҳҜдёӨеҗ‘йҮҸеӨ№и§’пјҢ\(\arcsin\rho = \pi/2 - \theta\)пјҢжүҮеҪўйқўз§Ҝ / еңҶйқўз§Ҝ е°ұжҳҜжҰӮзҺҮгҖӮ
05

з»Ҹе…ёйўҳпјҡиҒ”еҗҲжӯЈжҖҒжҰӮзҺҮи®Ўз®—

рҹҺҜ й«ҳйў‘йқўиҜ•йўҳ

и®ҫ \(X,Y\overset{iid}{\sim}\mathcal{N}(0,1)\)пјҢжұӮ \(P(X+Y>0 \text{ дё” } X+2Y>0)\)гҖӮ

Follow-up иӢҘ \(\mathrm{Cov}(X,Y)=\rho\)пјҢз»“жһңеҰӮдҪ•еҸҳеҢ–пјҹ

е®Ңж•ҙи§Јжі•

Step 1пјҡжһ„йҖ иҒ”еҗҲжӯЈжҖҒеҗ‘йҮҸ

д»Ө \(U=X+Y\)пјҢ\(V=X+2Y\)гҖӮеӣ  \((U,V)\) жҳҜ \((X,Y)\) зҡ„зәҝжҖ§еҸҳжҚўпјҢд»ҚжңҚд»ҺиҒ”еҗҲжӯЈжҖҒгҖӮ

$$\mathbb{E}[U]=\mathbb{E}[V]=0$$ $$\mathrm{Var}(U)=\mathrm{Var}(X)+\mathrm{Var}(Y)=2$$ $$\mathrm{Var}(V)=\mathrm{Var}(X)+4\mathrm{Var}(Y)=5$$ $$\mathrm{Cov}(U,V)=\mathrm{Cov}(X+Y,\;X+2Y)=\mathrm{Var}(X)+2\mathrm{Var}(Y)+3\mathrm{Cov}(X,Y)=1+2+0=3$$

Step 2пјҡи®Ўз®—зӣёе…ізі»ж•°

$$\rho_{UV} = \frac{\mathrm{Cov}(U,V)}{\sqrt{\mathrm{Var}(U)\mathrm{Var}(V)}} = \frac{3}{\sqrt{2\times5}} = \frac{3}{\sqrt{10}}$$

Step 3пјҡеә”з”Ёи§’еәҰе…¬ејҸ

$$P(U>0,V>0) = \frac{1}{4}+\frac{1}{2\pi}\arcsin\!\left(\frac{3}{\sqrt{10}}\right) \approx \frac{1}{4}+\frac{1}{2\pi}\cdot 1.2490 \approx \boxed{0.4488}$$

Follow-upпјҡ\(\mathrm{Cov}(X,Y)=\rho\)

$$\mathrm{Var}(U) = 2+2\rho, \quad \mathrm{Var}(V) = 5+4\rho$$ $$\mathrm{Cov}(U,V) = 3+3\rho$$ $$\rho_{UV} = \frac{3(1+\rho)}{\sqrt{(2+2\rho)(5+4\rho)}} = \frac{3\sqrt{1+\rho}}{\sqrt{2(5+4\rho)}} = \sqrt{\frac{9(1+\rho)}{2(5+4\rho)}}$$
$$P = \frac{1}{4}+\frac{1}{2\pi}\arcsin\!\sqrt{\frac{9(1+\rho)}{2(5+4\rho)}}$$
еҸҳдҪ“йўҳ

и®ҫ \(X,Y\overset{iid}{\sim}\mathcal{N}(0,1)\)пјҢ\(Z=X+Y\)пјҢжұӮ \(P(X>0\mid Z>0)\)гҖӮ

и§Јжі•
жіЁж„Ҹ \(\mathrm{Cov}(X,Z)=\mathrm{Var}(X)=1\)пјҢ\(\mathrm{Var}(Z)=2\)пјҢ\(\rho_{XZ}=\frac{1}{\sqrt{2}}\)гҖӮ
\(P(X>0,Z>0) = \frac{1}{4}+\frac{1}{2\pi}\arcsin\!\frac{1}{\sqrt{2}} = \frac{1}{4}+\frac{\pi/4}{2\pi}=\frac{3}{8}\)
\(P(X>0\mid Z>0) = \dfrac{P(X>0,Z>0)}{P(Z>0)} = \dfrac{3/8}{1/2} = \dfrac{3}{4}\)
06

\(\mathbb{E}[X \mid X > 0]\) зі»еҲ—

еҹәзЎҖз»“жһңжұҮжҖ»
жқЎд»¶жңҹжңӣз»“жһңж•°еҖј
\(\mathbb{E}[Z \mid Z>0]\)пјҢ\(Z\sim\mathcal{N}(0,1)\) \(\sqrt{2/\pi}\) \(\approx 0.7979\)
\(\mathbb{E}[Z^2 \mid Z>0]\) \(1\) \(= 1\)пјҲжіЁпјҡ\(\mathbb{E}[Z^2]=1\) дёҺжқЎд»¶ж— е…іпјү
\(\mathbb{E}[|Z|]\) \(\sqrt{2/\pi}\) \(\approx 0.7979\)
\(\mathbb{E}[Z \mid Z>c]\) \(\dfrac{\phi(c)}{1-\Phi(c)}\) Mills ratio
йқўиҜ•йўҳ

и®ҫ \(X,Y\overset{iid}{\sim}\mathcal{N}(0,1)\)пјҢе·ІзҹҘ \(X+Y>0\)пјҢжұӮ \(\mathbb{E}[X\mid X+Y>0]\)гҖӮ

и§Јжі•пјҲеҲ©з”ЁеҜ№з§°жҖ§ + Choleskyпјү
д»Ө \(Z=X+Y\sim\mathcal{N}(0,2)\)пјҢеҲҷ \(\rho_{XZ}=\frac{1}{\sqrt{2}}\)гҖӮ
жһ„йҖ пјҡ\(X = \frac{Z}{\sqrt{2}}\cdot\frac{1}{\sqrt{2}} + \sqrt{1-\frac{1}{2}}\cdot W = \frac{Z}{2} + \frac{W}{\sqrt{2}}\)пјҲе…¶дёӯ \(W\perp Z\)пјҢ\(W\sim\mathcal{N}(0,1)\)пјүгҖӮжӣҙзӣҙжҺҘең°пјҢ\(\mathbb{E}[X\mid Z] = \rho_{XZ}\frac{\sigma_X}{\sigma_Z}\cdot Z = \frac{1}{\sqrt{2}}\cdot\frac{1}{\sqrt{2}}\cdot Z = \frac{Z}{2}\)гҖӮ
\(\mathbb{E}[X\mid Z>0] = \mathbb{E}\!\left[\frac{Z}{2}\;\middle|\; Z>0\right] = \frac{1}{2}\mathbb{E}[Z\mid Z>0] = \frac{1}{2}\cdot\sqrt{2}\cdot\sqrt{\frac{2}{\pi}} = \frac{1}{\sqrt{\pi}}\)гҖӮ
$$\mathbb{E}[X \mid X+Y>0] = \frac{1}{\sqrt{\pi}} \approx 0.5642$$
рҹ’Ў
е…ій”®жӯҘйӘӨпјҡе…Ҳз”ЁжқЎд»¶жңҹжңӣзҡ„зәҝжҖ§жҖ§ \(\mathbb{E}[X\mid Z] = \frac{Z}{2}\)пјҲз”ұ Cov/Var еҫ—пјүпјҢеҶҚеҜ№дәӢд»¶ \(\{Z>0\}\) еҸ–жңҹжңӣгҖӮ
иҝӣйҳ¶йўҳ

и®ҫ \(X\sim\mathcal{N}(0,\sigma_X^2)\)пјҢ\(Y\sim\mathcal{N}(0,\sigma_Y^2)\) зӢ¬з«ӢпјҢжұӮ \(\mathbb{E}[X\mid X+Y>0]\)гҖӮ

и§Јжі•

д»Ө \(Z=X+Y\sim\mathcal{N}(0,\sigma_X^2+\sigma_Y^2)\)пјҢ\(\mathrm{Cov}(X,Z)=\sigma_X^2\)гҖӮ

$$\mathbb{E}[X\mid Z] = \frac{\sigma_X^2}{\sigma_X^2+\sigma_Y^2}\cdot Z$$
$$\mathbb{E}[X\mid Z>0] = \frac{\sigma_X^2}{\sigma_X^2+\sigma_Y^2}\cdot\mathbb{E}[Z\mid Z>0] = \frac{\sigma_X^2}{\sigma_X^2+\sigma_Y^2}\cdot\sqrt{\frac{2(\sigma_X^2+\sigma_Y^2)}{\pi}} = \sigma_X^2\sqrt{\frac{2}{\pi(\sigma_X^2+\sigma_Y^2)}}$$
07

\(\mathbb{E}[\max(X,Y)]\) зі»еҲ—

ж ёеҝғзӯүејҸ
$$\max(X,Y) = \frac{X+Y}{2} + \frac{|X-Y|}{2}$$

жҲ–зӯүд»·ең°пјҡ

$$\max(X,Y) + \min(X,Y) = X+Y, \qquad \max(X,Y)-\min(X,Y) = |X-Y|$$
з»Ҹе…ёйўҳ В· вӯҗвӯҗвӯҗ

и®ҫ \(X,Y\overset{iid}{\sim}\mathcal{N}(0,1)\)пјҢжұӮ \(\mathbb{E}[\max(X,Y)]\)гҖӮ

дёӨз§Қи§Јжі•

ж–№жі•дёҖпјҡеҲ©з”Ё \(X-Y\)

д»Ө \(W = X+Y\)пјҢ\(D = X-Y\)пјҢеҲҷ \(W\sim\mathcal{N}(0,2)\)пјҢ\(D\sim\mathcal{N}(0,2)\)пјҢдё” \(W\perp D\)пјҲиҒ”еҗҲжӯЈжҖҒдёӯдёҚзӣёе…іеҚізӢ¬з«ӢпјүгҖӮ
\(\max(X,Y) = \frac{W+|D|}{2}\)пјҢж•… \(\mathbb{E}[\max(X,Y)] = \frac{\mathbb{E}[W]+\mathbb{E}[|D|]}{2} = \frac{0+\mathbb{E}[|D|]}{2}\)гҖӮ
\(\mathbb{E}[|D|] = \mathbb{E}[|D|]\)пјҢ\(D\sim\mathcal{N}(0,2)\)пјҢж•… \(\mathbb{E}[|D|] = \sqrt{2}\cdot\mathbb{E}[|Z|] = \sqrt{2}\cdot\sqrt{2/\pi} = 2/\sqrt{\pi}\)гҖӮ
$$\mathbb{E}[\max(X,Y)] = \frac{1}{\sqrt{\pi}} \approx 0.5642$$

ж–№жі•дәҢпјҡзӣҙжҺҘз§ҜеҲҶпјҲж¬ЎеәҸз»ҹи®ЎйҮҸпјү

$$\mathbb{E}[\max(X,Y)] = \int_{-\infty}^{\infty} x\cdot 2\Phi(x)\phi(x)\,dx = 2\int_{-\infty}^{\infty} x\Phi(x)\phi(x)\,dx$$

еҲҶйғЁз§ҜеҲҶд»Ө \(u=\Phi(x)\)пјҢ\(dv=2x\phi(x)dx = -2d\phi(x)\)пјҡ

$$= -2\phi(x)\Phi(x)\big|_{-\infty}^{\infty} + 2\int\phi(x)^2\,dx = 0 + 2\cdot\frac{1}{2\sqrt{\pi}} = \frac{1}{\sqrt{\pi}}$$
иҝӣйҳ¶йўҳ

и®ҫ \((X,Y)\sim\mathcal{N}(0,I)\)пјҲдәҢз»ҙж ҮеҮҶпјҢзӣёе…ізі»ж•° \(\rho\)пјүпјҢжұӮ \(\mathbb{E}[\max(X,Y)]\)гҖӮ

и§Јжі•
\(W=X+Y\sim\mathcal{N}(0,2+2\rho)\)пјҢ\(D=X-Y\sim\mathcal{N}(0,2-2\rho)\)пјҢдё” \(W\perp D\)гҖӮ
\(\mathbb{E}[\max(X,Y)] = \frac{1}{2}\mathbb{E}[|D|] = \frac{1}{2}\cdot\sqrt{2-2\rho}\cdot\sqrt{\frac{2}{\pi}} = \sqrt{\frac{1-\rho}{\pi}}\)
$$\mathbb{E}[\max(X,Y)] = \sqrt{\frac{1-\rho}{\pi}}$$

йӘҢиҜҒпјҡ\(\rho=0\) ж—¶ \(=1/\sqrt\pi\) вң“пјӣ\(\rho=1\) ж—¶ \(=0\)пјҲдёӨеҸҳйҮҸзӣёеҗҢпјҢmax=XпјҢ\(\mathbb{E}=0\)пјүвң“пјӣ\(\rho=-1\) ж—¶ \(=\sqrt{2/\pi}\)пјҲmax(X,-X)=|X|пјҢ\(\mathbb{E}=\sqrt{2/\pi}\)пјүвң“гҖӮ

еҸҳдҪ“пјҡ\(\mathbb{E}[\max(0,Z)]\)
и§Јжі•
$$\max(0,Z) = Z\cdot\mathbf{1}_{Z>0} = \frac{Z+|Z|}{2}$$
$$\mathbb{E}[\max(0,Z)] = \frac{0+\mathbb{E}[|Z|]}{2} = \frac{1}{2}\cdot\sqrt{\frac{2}{\pi}} = \frac{1}{\sqrt{2\pi}}$$

йҮ‘иһҚеҗ«д№үпјҡиҝҷжӯЈжҳҜж ҮеҮҶжӯЈжҖҒдёӢ call option зҡ„ vega зӣёе…ійҮҸгҖӮ

08

жқЎд»¶жңҹжңӣз»јеҗҲпјҡ\(\mathbb{E}[X \mid X + Y > 0]\)

йҖҡз”ЁжЎҶжһ¶

еҜ№дәҺиҒ”еҗҲжӯЈжҖҒ \((X,Y)\)пјҢи®Ўз®— \(\mathbb{E}[X\mid aX+bY>0]\) зҡ„ж ҮеҮҶжөҒзЁӢпјҡ

д»Ө \(Z=aX+bY\)пјҢи®Ўз®— \(\mathrm{Cov}(X,Z)\) е’Ң \(\mathrm{Var}(Z)\)гҖӮ
еҲ©з”ЁиҒ”еҗҲжӯЈжҖҒзҡ„жҠ•еҪұпјҡ\(\mathbb{E}[X\mid Z] = \mathbb{E}[X] + \frac{\mathrm{Cov}(X,Z)}{\mathrm{Var}(Z)}(Z-\mathbb{E}[Z])\)гҖӮ
з”Ёе…Ёжңҹжңӣпјҡ\(\mathbb{E}[X\mid Z>0] = \mathbb{E}[\mathbb{E}[X\mid Z]\mid Z>0]\)гҖӮ
жңҖз»ҲеҪ’з»“дёә \(\mathbb{E}[Z\mid Z>0]\) зҡ„и®Ўз®—гҖӮ
з»јеҗҲйўҳ

и®ҫ \(X,Y\overset{iid}{\sim}\mathcal{N}(0,1)\)пјҢ\(Z = X+Y\)гҖӮ

(a) жұӮ \(\mathbb{E}[X\mid Z]\)пјӣ

(b) жұӮ \(\mathbb{E}[X\mid Z>0]\)пјӣ

(c) жұӮ \(\mathbb{E}[Z\mid X>0]\)гҖӮ

и§Јзӯ”

(a) з”ұиҒ”еҗҲжӯЈжҖҒжқЎд»¶жңҹжңӣе…¬ејҸпјҡ

$$\mathbb{E}[X\mid Z=z] = 0 + \frac{\mathrm{Cov}(X,Z)}{\mathrm{Var}(Z)}(z-0) = \frac{1}{2}z \implies \mathbb{E}[X\mid Z] = \frac{Z}{2}$$

(b) \(\mathbb{E}[X\mid Z>0] = \mathbb{E}\left[\frac{Z}{2}\;\middle|\;Z>0\right] = \frac{1}{2}\mathbb{E}[Z\mid Z>0] = \frac{1}{2}\cdot\sqrt{2}\cdot\sqrt{\frac{2}{\pi}} = \dfrac{1}{\sqrt\pi}\)гҖӮ

(c) з”ұеҜ№з§°жҖ§пјҢ\(\mathbb{E}[Z\mid X>0]=\mathbb{E}[X+Y\mid X>0] = \mathbb{E}[X\mid X>0]+\mathbb{E}[Y\mid X>0] = \sqrt{2/\pi}+0 = \sqrt{2/\pi}\)пјҲ\(Y\perp X\)пјүгҖӮ

Box-Muller еҸҳжҚў

й—®йўҳпјҡеҰӮдҪ•д»Һ \(U(0,1)\) з”ҹжҲҗ \(\mathcal{N}(0,1)\)

йҖҶеҸҳжҚўжі•дёҚзӣҙжҺҘеҸҜиЎҢпјҲ\(\Phi^{-1}\) ж— й—ӯеҗҲеҪўејҸпјүпјҢBox-Muller еҲ©з”ЁжһҒеқҗж ҮеҸҳжҚўз»•иҝҮиҝҷдёҖеӣ°йҡҫгҖӮ

з®—жі•
йҮҮж ·и§’еәҰпјҡ\(\Theta \sim \text{Uniform}[0, 2\pi)\)гҖӮд»Ө \(U_1\sim U(0,1)\)пјҢеҸ– \(\Theta = 2\pi U_1\)гҖӮ
йҮҮж ·еҚҠеҫ„пјҡйңҖиҰҒ \(R\) дҪҝеҫ— \((X,Y)=(R\cos\Theta, R\sin\Theta)\) жҳҜж ҮеҮҶдәҢз»ҙжӯЈжҖҒгҖӮ
дәҢз»ҙж ҮеҮҶжӯЈжҖҒзҡ„еҫ„еҗ‘еҲҶеёғпјҡ\(f_R(r)=re^{-r^2/2}\)пјҲRayleigh еҲҶеёғпјүпјҢCDF = \(1-e^{-r^2/2}\)гҖӮ
йҖҶеҸҳжҚўпјҡд»Ө \(U_2\sim U(0,1)\)пјҢеҲҷ \(R = \sqrt{-2\ln U_2}\)гҖӮ
иҫ“еҮәдёӨдёӘзӢ¬з«Ӣж ҮеҮҶжӯЈжҖҒпјҡ
$$X = \sqrt{-2\ln U_2}\cos(2\pi U_1), \qquad Y = \sqrt{-2\ln U_2}\sin(2\pi U_1)$$
рҹ“җ
е…ій”®жҺЁеҜјпјҡ\(\iint e^{-(x^2+y^2)/2}dxdy = \int_0^{2\pi}\int_0^\infty e^{-r^2/2}r\,dr\,d\theta = 2\pi\)пјҢж•…иҒ”еҗҲ pdf еҸҜеҲҶзҰ»дёәи§’еәҰпјҲеқҮеҢҖпјүе’ҢеҚҠеҫ„пјҲRayleighпјүдёӨйғЁеҲҶгҖӮ
Marsaglia жһҒеқҗж Үжі•пјҲBox-Muller зҡ„ж”№иҝӣпјү

йҒҝе…Қдёүи§’еҮҪж•°и®Ўз®—пјҡ

еқҮеҢҖйҮҮж ·еҚ•дҪҚеңҶеҶ…зҡ„зӮ№пјҡйҮҮж · \((u,v)\in[-1,1]^2\)пјҢжӢ’з»қ \(s=u^2+v^2\geq 1\)гҖӮ
иҫ“еҮә \(X=u\sqrt{-2\ln s/s}\)пјҢ\(Y=v\sqrt{-2\ln s/s}\)гҖӮ

жңҹжңӣжҺҘеҸ—зҺҮ = \(\pi/4 \approx 78.5\%\)пјҢйҒҝе…ҚдәҶ \(\cos,\sin\) зҡ„и®Ўз®—гҖӮ

Cholesky еҲҶи§Јпјҡз”ҹжҲҗзӣёе…іжӯЈжҖҒ

й—®йўҳи®ҫе®ҡ

е·ІзҹҘ \(Z_1,\ldots,Z_n\overset{iid}{\sim}\mathcal{N}(0,1)\)пјҢеҰӮдҪ•жһ„йҖ  \(\mathbf{X}\sim\mathcal{N}(\boldsymbol{\mu},\Sigma)\)пјҹ

Cholesky еҲҶи§Јжі•
еҜ№жӯЈе®ҡзҹ©йҳө \(\Sigma\) еҒҡ Cholesky еҲҶи§Јпјҡ\(\Sigma = LL^T\)пјҢ\(L\) жҳҜдёӢдёүи§’зҹ©йҳөгҖӮ
д»Ө \(\mathbf{Z}=(Z_1,\ldots,Z_n)^T\sim\mathcal{N}(\mathbf{0},I)\)гҖӮ
еҸ– \(\mathbf{X} = \boldsymbol{\mu} + L\mathbf{Z}\)пјҢеҲҷ \(\mathbb{E}[\mathbf{X}]=\boldsymbol{\mu}\)пјҢ\(\mathrm{Cov}(\mathbf{X})=LL^T=\Sigma\)гҖӮ
дәҢз»ҙжҳҫејҸжһ„йҖ пјҲ\(n=2\)пјү
$$L = \begin{pmatrix}\sigma_X & 0\\\rho\sigma_Y & \sqrt{1-\rho^2}\,\sigma_Y\end{pmatrix}$$
$$X = \mu_X + \sigma_X Z_1, \qquad Y = \mu_Y + \rho\sigma_Y Z_1 + \sqrt{1-\rho^2}\,\sigma_Y Z_2$$

еҸҜд»ҘйӘҢиҜҒ \(\mathrm{Corr}(X,Y)=\rho\)пјҢдё” \(Y\mid X=x\) дёӯ \(X\) еӣәе®ҡеҗҺпјҢйҡҸжңәжҖ§д»…жқҘиҮӘ \(Z_2\)пјҢж–№е·®дёә \((1-\rho^2)\sigma_Y^2\)гҖӮ

рҹ’Ў
жқЎд»¶еҲҶеёғжҺЁеҜјзҡ„еҮ дҪ•ж„Ҹд№үпјҡ\(X\) еҶіе®ҡдәҶ \(Z_1\)пјҢ\(Y\mid X\) зҡ„еҲҶеёғжқҘиҮӘ \(Z_2\) зҡ„йғЁеҲҶпјҢеҚі \(\rho\) жҚ•жҚүдәҶ"е·ІзҹҘ"зҡ„дҝЎжҒҜйҮҸпјҢеү©дҪҷдёҚзЎ®е®ҡжҖ§дёә \(1-\rho^2\)гҖӮ
жһ„йҖ йўҳ

з»ҷе®ҡ \(X,Y,Z\) дёӨдёӨзӢ¬з«ӢеқҮдёә \(\mathcal{N}(0,1)\)пјҢеҰӮдҪ•жһ„йҖ дёҖеҜ№зӣёе…ізі»ж•°жҒ°еҘҪдёә \(1/2\) зҡ„иҒ”еҗҲжӯЈжҖҒеҜ№ \((A,B)\)пјҹ

и§Јжі•
$$A = X, \qquad B = \frac{X + \sqrt{3}Y}{2}$$

йӘҢиҜҒпјҡ\(\mathrm{Var}(B)=\frac{1+3}{4}=1\)пјҢ\(\mathrm{Cov}(A,B)=\frac{1}{2}\)пјҢ\(\rho=\frac{1/2}{1\cdot 1}=\frac{1}{2}\)гҖӮвң“

11

Shrinkage EstimatorпјҲж”¶зј©дј°и®ЎйҮҸпјү

з»Ҹе…ёи®ҫе®ҡ

и®ҫ \(X\sim\mathcal{N}(0,\sigma_X^2)\) дёәзңҹе®һдҝЎеҸ·пјҲжңӘзҹҘпјүпјҢ\(\varepsilon\sim\mathcal{N}(0,\sigma_\varepsilon^2)\) дёәеҷӘеЈ°пјҲзӢ¬з«ӢпјүпјҢи§ӮжөӢ \(Z=X+\varepsilon\)гҖӮз»ҷе®ҡи§ӮжөӢ \(Z=z\)пјҢеҰӮдҪ•дј°и®Ў \(X\)пјҹ

MMSE дј°и®ЎжҺЁеҜј

зӣҙи§үпјҡ\(Z\) жҳҜ \(X\) зҡ„еҗ«еҷӘзүҲжң¬пјҢжңҖдјҳдј°и®ЎдёҚжҳҜзӣҙжҺҘз”Ё \(Z\) жң¬иә«пјҢиҖҢжҳҜжҢүдҝЎеҷӘжҜ”"ж”¶зј©"гҖӮ

и®Ўз®—з»ҹи®ЎйҮҸпјҡ\(\mathbb{E}[Z]=0\)пјҢ\(\mathrm{Var}(Z)=\sigma_X^2+\sigma_\varepsilon^2\)пјҢ\(\mathrm{Cov}(X,Z)=\sigma_X^2\)гҖӮ
и®Ўз®—зӣёе…ізі»ж•°пјҡ\(\rho_{XZ} = \dfrac{\sigma_X^2}{\sigma_X\sqrt{\sigma_X^2+\sigma_\varepsilon^2}} = \dfrac{\sigma_X}{\sqrt{\sigma_X^2+\sigma_\varepsilon^2}}\)гҖӮ
з”ұиҒ”еҗҲжӯЈжҖҒжқЎд»¶жңҹжңӣе…¬ејҸпјҡ
$$\mathbb{E}[X\mid Z=z] = \frac{\sigma_X^2}{\sigma_X^2+\sigma_\varepsilon^2}\cdot z = \frac{\mathrm{SNR}}{1+\mathrm{SNR}}\cdot z$$
е…¶дёӯ \(\mathrm{SNR}=\sigma_X^2/\sigma_\varepsilon^2\)гҖӮ
жқЎд»¶ж–№е·®пјҡ\(\mathrm{Var}(X\mid Z=z) = \sigma_X^2(1-\rho_{XZ}^2) = \dfrac{\sigma_X^2\sigma_\varepsilon^2}{\sigma_X^2+\sigma_\varepsilon^2}\)гҖӮ
рҹ“Ҡ
Shrinkage зі»ж•° \(\lambda = \dfrac{\sigma_X^2}{\sigma_X^2+\sigma_\varepsilon^2}\in[0,1]\)пјҡ
вҖў \(\sigma_\varepsilon^2\to 0\)пјҲж— еҷӘеЈ°пјүпјҡ\(\lambda\to 1\)пјҢзӣҙжҺҘдҝЎд»»и§ӮжөӢеҖјпјӣ
вҖў \(\sigma_\varepsilon^2\to\infty\)пјҲе…ЁеҷӘеЈ°пјүпјҡ\(\lambda\to 0\)пјҢж”¶зј©еҲ°е…ҲйӘҢеқҮеҖјпјҲ0пјүпјӣ
вҖў иҝҷе°ұжҳҜ James-Stein ж”¶зј© зҡ„жң¬иҙЁпјҢд№ҹжҳҜ Kalman Filter жӣҙж–°жӯҘзҡ„еҹәзЎҖгҖӮ
еҸҳдҪ“пјҡеӨҡж¬Ўи§ӮжөӢ

и®ҫ \(Z_1,\ldots,Z_n\) жҳҜ \(X\) зҡ„ iid еҗ«еҷӘи§ӮжөӢпјҲеҗҢж–№е·® \(\sigma_\varepsilon^2\)пјүпјҢ\(\bar{Z}=\frac{1}{n}\sum Z_i\)пјҢжұӮ \(\mathbb{E}[X\mid Z_1,\ldots,Z_n]\)гҖӮ

и§Јзӯ”

е……еҲҶз»ҹи®ЎйҮҸдёә \(\bar{Z}\sim\mathcal{N}(0,\sigma_X^2+\sigma_\varepsilon^2/n)\)пјҢ\(\mathrm{Cov}(X,\bar{Z})=\sigma_X^2\)пјҢеӣ жӯӨпјҡ

$$\mathbb{E}[X\mid Z_1,\ldots,Z_n] = \frac{\sigma_X^2}{\sigma_X^2+\sigma_\varepsilon^2/n}\cdot\bar{Z}$$

йҡҸзқҖ \(n\to\infty\)пјҢзі»ж•°и¶Ӣиҝ‘дәҺ 1пјҲж— йҷҗи§ӮжөӢдёӢжҒўеӨҚзңҹе®һеҖјпјүгҖӮ

12

з»јеҗҲйўҳеә“

йўҳ A вӯҗвӯҗ

и®ҫ \(X,Y\overset{iid}{\sim}\mathcal{N}(0,1)\)пјҢжұӮ \(P(X^2+Y^2 \leq 1)\)гҖӮ

и§Јзӯ”

\(R^2 = X^2+Y^2\sim\chi^2(2) = \text{Exp}(1/2)\)пјҲеҚі \(P(R^2\leq r) = 1-e^{-r/2}\)пјүгҖӮ

$$P(X^2+Y^2\leq 1)=1-e^{-1/2}\approx 0.3935$$
йўҳ B вӯҗвӯҗ

и®ҫ \(X\sim\mathcal{N}(0,1)\)пјҢжұӮ \(\mathbb{E}[e^{tX}]\)пјҲзҹ©з”ҹжҲҗеҮҪж•°пјүгҖӮ

и§Јзӯ”
$$M_X(t) = \mathbb{E}[e^{tX}] = e^{t^2/2}$$

жҺЁеҜјпјҡй…Қж–№ \(-\frac{x^2}{2}+tx = -\frac{(x-t)^2}{2}+\frac{t^2}{2}\)пјҢз§ҜеҲҶеҪ’дёҖеҢ–еҗҺеҫ— \(e^{t^2/2}\)гҖӮ

жҺЁе№ҝпјҡиӢҘ \(X\sim\mathcal{N}(\mu,\sigma^2)\)пјҢеҲҷ \(M_X(t)=e^{\mu t+\sigma^2t^2/2}\)гҖӮ

йўҳ C вӯҗвӯҗвӯҗ

и®ҫ \(X,Y,Z\overset{iid}{\sim}\mathcal{N}(0,1)\)пјҢдёӨдёӨзӣёе…ізі»ж•°еқҮдёә \(1/2\)пјҢжұӮ \(P(X+Y+Z>0)\)гҖӮ

и§Јзӯ”

\(W=X+Y+Z\sim\mathcal{N}(0, \mathrm{Var}(W))\)гҖӮ

\(\mathrm{Var}(W) = 3\cdot 1 + 2\cdot 3\cdot\frac{1}{2} = 3+3=6\)гҖӮ

ж•… \(W\sim\mathcal{N}(0,6)\)пјҢз”ұеҜ№з§°жҖ§пјҡ

$$P(W>0) = \frac{1}{2}$$
йўҳ D вӯҗвӯҗвӯҗ

и®ҫ \(X\sim\mathcal{N}(\mu,\sigma^2)\)пјҢ\(\mu>0\)пјҢжұӮ \(\mathbb{E}[X\mid X>0]\)гҖӮ

и§Јзӯ”пјҲMills Ratio е…¬ејҸпјү
$$\mathbb{E}[X\mid X>0] = \mu + \sigma\cdot\frac{\phi(\mu/\sigma)}{\Phi(\mu/\sigma)}$$

еҪ“ \(\mu=0\)пјҡ\(\mathbb{E}[X\mid X>0]=\sigma\frac{\phi(0)}{\Phi(0)}=\sigma\cdot\frac{1/\sqrt{2\pi}}{1/2}=\sigma\sqrt{2/\pi}\) вң“гҖӮ

йўҳ E вӯҗвӯҗвӯҗ

и®ҫ \(X\sim\mathcal{N}(0,1)\)пјҢз”Ё \(Y=\text{sgn}(X)\cdot|X|\) жқҘжһ„йҖ дёӨдёӘиҫ№зјҳжӯЈжҖҒдҪҶиҒ”еҗҲйқһжӯЈжҖҒзҡ„йҡҸжңәеҸҳйҮҸпјҢе№¶и®Ўз®— \(\mathrm{Cov}(X,Y)\)гҖӮ

и§Јзӯ”

еҸ– \(W\sim\pm 1\)пјҲзӢ¬з«ӢдәҺ \(X\)пјҢеҗ„ \(1/2\)пјүпјҢд»Ө \(Y=W\cdot X\)гҖӮ

вҖў \(Y\sim\mathcal{N}(0,1)\)пјҲеӣ дёә \(\pm X\) еқҮдёәж ҮеҮҶжӯЈжҖҒпјүгҖӮ

вҖў \(\mathrm{Cov}(X,Y)=\mathbb{E}[XY]=\mathbb{E}[WX^2]=\mathbb{E}[W]\mathbb{E}[X^2]=0\)гҖӮ

вҖў дҪҶ \(X+Y=X(1+W)\)пјҢеҪ“ \(W=-1\) ж—¶жҒ’дёә 0пјҢж•… \(X+Y\) дёҚжҳҜжӯЈжҖҒеҲҶеёғпјҢ\((X,Y)\) дёҚжҳҜиҒ”еҗҲжӯЈжҖҒгҖӮ

вҡ пёҸ
иҝҷжҳҜ"иҫ№зјҳжӯЈжҖҒ вү  иҒ”еҗҲжӯЈжҖҒ"зҡ„з»Ҹе…ёеҸҚдҫӢпјҢйқўиҜ•дёӯз»Ҹеёёиў«з”ЁжқҘиҖғеҜҹзҗҶи§Јж·ұеәҰгҖӮ
йўҳ F вӯҗвӯҗвӯҗвӯҗ HFTйЈҺж ј
